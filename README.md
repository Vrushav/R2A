# R2A – Rural Reach Appointment Access

🚀 **Live Demo:** https://r2a.onrender.com

R2A is a rural-first healthcare platform designed to improve healthcare accessibility in underserved communities.

## Vision

To bridge the gap between rural citizens and healthcare services through a simple, scalable, and technology-driven platform.

## Core Features

- Patient Appointment Booking
- Family Health Card
- AI Health Assistant
- Hospital & Doctor Dashboards
- ASHA Worker Support
- Medicine Availability
- Queue Management
- Offline Access
- Multilingual Interface
- QR Appointment Pass

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- Git & GitHub

## Project Structure

- frontend/ - user-facing web app
  - public/ - static entry point and assets
    - index.html
    - assets/css/
    - assets/js/
    - assets/images/
- backend/ - server and API logic
  - server.js
  - tests/
  - data/
- docs/ - planning and architecture documentation
- database/ - database design and schema files

## Getting Started

1. Install dependencies:
   npm install
2. Start the backend server:
   node backend/server.js
3. Open the frontend entry file in the browser or serve the frontend folder with a static server.
