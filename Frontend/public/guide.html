R2A User Guide

R2A helps patients book government healthcare appointments in a simple, multilingual experience.

Open the App

How to run the app

  1. Open the project folder.
  2. Double-click launch.bat or run node server.js.
  3. Open http://localhost:3000 in the browser.

How users interact

  • Register as a patient and enter basic details.
  • Choose a village, hospital, department, and doctor.
  • Book an appointment and receive a confirmation slip.
  • Use the AI assistant for department guidance.
  • Access hospital and doctor dashboards for queue tracking.

Tips for smooth use

  • Use a modern browser such as Chrome or Edge.
  • Allow location and browser permissions if prompted.
  • Make sure the app is running before opening it.
